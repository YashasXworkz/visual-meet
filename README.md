# Visual Meeting

A real-time video conferencing web application.

## Getting Started

1. Clone the repository
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies
   ```sh
   cd <YOUR_PROJECT_NAME>
   npm i
   ```

3. Start development server
   ```sh
   npm run dev
   ```

## Technology Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Node.js/Express backend
- MongoDB

## Features

- Video conferencing
- User authentication
- Real-time chat
